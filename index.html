À propos

Un peu plus sur moi

Développeur web formé chez OpenClassrooms, je travaille principalement avec WordPress ainsi qu'avec les fondamentaux du web : HTML, CSS et JavaScript.

J'aime partir d'une maquette pour la rendre vivante : intégration fidèle, animations discrètes, code lisible et respect des bonnes pratiques d'accessibilité. Chaque projet est l'occasion d'apprendre quelque chose de nouveau.

En dehors du code, je continue à explorer de nouveaux outils et à peaufiner mes projets personnels disponibles sur GitHub.
