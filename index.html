À propos

Un peu plus sur moi

Développeur web formé chez OpenClassrooms, je construis des sites avec WordPress (thèmes, thèmes enfants, Elementor) tout en gardant les mains dans les fondamentaux : HTML, CSS et JavaScript.

Ma façon de travailler est simple : partir d'une maquette et la rendre vivante au pixel près — intégration fidèle, animations discrètes, code lisible et accessible. Je crois qu'un bon site doit être aussi agréable à utiliser qu'à maintenir.

Curieux de nature, je continue d'apprendre à chaque projet et de peaufiner mes réalisations personnelles, toutes ouvertes sur GitHub.
